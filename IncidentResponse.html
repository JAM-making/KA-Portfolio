Detecting and Cleaning Up Malicious Processes

Week 4 | M10E Scenario: Stop Ransomware Attack – Operation Crimson Wolf

Anna Kephart | ITS4239 – Incident Response SU01 | Instructor: Dr. Meadors | October 11, 2025

Introduction

Objective 5 required identifying and stopping malicious processes on a compromised host. Tools used included VNC, Remote Desktop Protocol, and Process Explorer to investigate suspicious activity and confirm the presence of WannaCry-related files and processes.

Malicious executables identified included av.exe, mssecsvc2.0, and tasksche.exe, which mimicked legitimate system processes. By analyzing process trees, examining hidden file paths, and inspecting system internals, the malicious services were terminated and the infection contained.

This task demonstrates practical incident response skills, including log analysis, system administration commands, and malware verification, reflecting real-world ransomware response procedures.

Investigation and Process Analysis

Accessing the infected machine via VNC and RDP, I located the suspicious IP 172.16.10.102 from prior logs in ELSA. Initial exploration revealed signs of system corruption.

Using Sysinternals and Process Explorer, I examined all running processes, including parent/child relationships, loaded DLLs, network connections, CPU/memory usage, and file locations. This allowed the identification of suspicious processes such as mssecsvc2.0, tasksche.exe, and av.exe.

Hovering over each process revealed the full file paths. Malicious processes were terminated using procexp.exe and sc stop mssecsvc2.0 to contain the WannaCry infection and prevent further spread.

Malware Cleanup Steps

  1. Deleted the @WanaDecryptor@ file from the local C drive.
  2. Opened Process Explorer as Administrator to view all processes.
  3. Identified suspicious processes and their hidden file paths.
  4. Terminated malicious processes and their child processes.
  5. Documented corrupted and hidden paths for further system hardening.

Relevance to Cybersecurity

This exercise highlights core cybersecurity skills: process forensics, malware containment, and system hardening. It demonstrates the ability to analyze, identify, and respond to live threats in an enterprise environment, mirroring real-world incident response.
